# 🎬 AI Video Ads — Social Eagle Kids

A short-form AI-generated advertisement showcasing how kids can start learning AI in a practical and engaging way.

---

## 📖 Project Overview

**AI Video Ads — Social Eagle Kids** is a short-form video concept created entirely using AI tools.  
The project demonstrates a realistic storytelling approach — starting from a family scenario and transitioning into a TV advertisement that introduces AI learning for kids.

The goal is to present AI education in a **relatable, natural, and engaging format**, making it easy for parents and kids to understand its importance today.

## 🛠️ AI Tools Used

### 🎥 Video Generation
- **Google Flow (VEO 3.1 Fast)** — Primary video generation  

### 🧠 Prompt Engineering
- **ChatGPT** — Scene design, dialogue structuring, and prompt creation  

### 🎬 Post Production
- **Clipchamp** — Final edit, logo overlay, and export  

---

## 🚀 Getting Started

### 1. Prepare Prompts
Create structured JSON prompts for each scene using ChatGPT.

### 2. Generate Video
Use prompts in **Google Flow (VEO 3.1 Fast)** to generate clips.

### 3. Combine Clips
Import clips into Clipchamp and arrange sequentially.

### 4. Add Branding
- Add logo (top-right overlay)  
- Include CTA screen  

### 5. Export Final Video
Export in **1080p (MP4 format)**
